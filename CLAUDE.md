# NoveuFlow Ecosystem

Meta-repository voor het NoveuFlow ecosystem. Bevat geen code, alleen overzicht en tooling.

## Repos

- **noveuflow-wordpress** — WordPress plugin (PHP + React SPA). Actieve ontwikkeling.
- **marketing-website-noveuflow** — Next.js marketing site, deployed via GitHub Actions naar Cloudflare Pages. NIET AANRAKEN zonder overleg.
- **noveuflow-saas** — Toekomstige standalone SaaS versie.
- **noveuflow-npm** — Toekomstig NPM package voor embedden.
- **noveuflow-odoo** — Toekomstige Odoo integratie.
- **noveuflow-archief** — Legacy code (Mobiliteit-Connect, Efficio). Alleen ter referentie.
